WESTERN CRETE COMPANION - GITHUB PAGES DEPLOYMENT

1. Create a free GitHub account at github.com if needed.
2. Create a new PUBLIC repository, for example: western-crete-companion.
3. Upload every file and folder from this package to the repository root.
4. Open Settings > Pages.
5. Under Build and deployment choose: Deploy from a branch.
6. Select branch: main, folder: /(root), then Save.
7. Wait 1-3 minutes. GitHub will show the web address.
8. Open that address in Safari on the iPhone.
9. Tap Share > Add to Home Screen.

IMPORTANT
- Photos are loaded from Unsplash, so the first visit needs internet access. The service worker then caches files and viewed images where Safari allows.
- Opening index.html directly from Files will not give the same app behaviour. Use the GitHub Pages address.
- TripAdvisor buttons use a TripAdvisor search link because individual listing URLs can change.
- Check opening hours and prices on official sites immediately before travel.
